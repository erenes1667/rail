# Rail

Stripped-down Amphetamine for Apple Silicon Macs. Menu-bar app that keeps macOS awake on demand, with a syringe-into-bicep animation that settles into a continuous flex while active.

![menu bar pill → inject animation → flexing bicep]

## Why

Apple Silicon enforces lid-close sleep at the firmware level. `caffeinate` and `IOPMAssertion`-only tools (Amphetamine, KeepingYouAwake, etc.) gate the **idle** and **display** sleep paths, but not the lid switch. The only software-level escape is `sudo pmset -a disablesleep 1`.

Amphetamine has the right UI but ships with features I don't need. Rail is the same shape minus the noise: pill icon, click to toggle, timer presets, three checkboxes that persist across launches.

## Interaction

- **Right-click pill** → toggle ON / OFF. On turn-on: syringe descends, plunger pushes (~0.85s), drip + jiggle, syringe yanks out, comedic bicep flex pop with sparkles.
- **Left-click pill** → opens the options menu. Stays open until click-away or pill re-click.
- **Active state** → bicep keeps pulsing on a 2.4s cycle (fast pose-up, slow release). Sparkles fire at the apex.

Checkbox state persists via `UserDefaults` and survives reboots.

## Under the hood

| Mechanism | Purpose | Gated by |
|---|---|---|
| `IOPMAssertionCreateWithName("PreventUserIdleSystemSleep")` | Block idle system sleep | always-on while active |
| `IOPMAssertionCreateWithName("PreventUserIdleDisplaySleep")` | Block idle display sleep | "Allow display sleep" checkbox |
| `sudo -n pmset -a disablesleep 1` | Block lid-close sleep | "Allow system sleep when lid is closed" checkbox |
| `IOPMAssertionDeclareUserActivity` every 30s | Reset user-idle counter | "Allow screen saver after 45m" checkbox |

Toggling a checkbox while active hot-refreshes the assertions.

> Requires passwordless `sudo` for `pmset`. Without it the lid-close behavior won't engage but everything else still works.

## App icon

Procedurally generated by `MakeIcon.swift` at build time. Dark mirrored-glass background, two diagonal white powder rails with scattered grain texture, a small syringe across the upper-right corner. Vignette + top sheen for depth. Generated at 1024² and packed into multi-resolution `Rail.icns` (16 → 1024).

To redesign the icon: edit `MakeIcon.swift` (bezier-path drawing), then `./build.sh`.

## Build + install

```bash
./build.sh --launch
```

Installs to `/Applications/Rail.app` and clears the LaunchServices icon cache so the new icon shows up immediately.

## Requirements

- macOS 13+
- Apple Silicon (the lid-close override is Apple-Silicon-specific)
- Xcode command-line tools (`xcode-select --install`) for `swiftc`
- Passwordless `sudo` for the lid-close behavior (optional but recommended)
